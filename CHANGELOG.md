# Matchkeeper — Changelog

سجل تراكمي بكل إصدارات Matchkeeper والتحديثات اللي كل إصدار شملها. يتحدث تلقائي كل مرة يصدر فيها إصدار جديد.

## V0.04.09 (الحالي)

- **Break Preference — Anchor حقيقي للراوند الأول:** أي لاعب بريفرنسه "Early" بيتحجزله مكان بالأولوية في بريك الراوند الأول، بغض النظر عن الاستحقاق (مش مجرد تفضيل عند التعادل زي قبل كده)
- **⚖️ ميزان توازن + زرار Re-pair (الراوند الأول بس):** بيوري الفرق في USR بين فريقي أي ملعب، ومعاه زرار لإعادة توزيع نفس الأربعة لاعبين بالتوازن لو الأدمن بدّل حد من/للبريك
- **C1/C2 (الملعب المتوقع) للبريك في كل الراوندات:** كانت موجودة للراوند الأول بس، دلوقتي بتظهر لأي حد في البريك في أي راوند، محسوبة من آخر نتيجة حقيقية سجّلها (فوز يرقّي، خسارة تنزّل)

## V0.04.08

**من V0.04.00 لحد دلوقتي:**

### Break Engine
- Break Preference ظاهرة في Profile Screen (بدون Edit) + قابلة للتعديل من هناك
- Break Preference متاحة في فورم Platform Admin (Add/Edit User)
- قابلة للتعديل في Event → Players tab أي وقت (قبل أو بعد الـ plan)
- أولوية البريفرنس في خوارزمية التوزيع اتزودت (بقت العامل التاني بعد الاستحقاق، مش آخر حاجة)

### استقرار البيانات
- إصلاح جذري لمشكلة "الداتا الناقصة / 12 لاعب بس" — القراءة من Firestore بقت بتستنى تسجيل الدخول يخلص فعلياً قبل ما تبدأ
- تحذير تشخيصي على الشاشة بيوري الـ error الحقيقي (كود + رسالة) لو أي جزء من الداتا فشل يتحمل

### Match Mode + الإشعارات
- تايمر Match Mode بياخد الوقت الحقيقي اللي الأدمن بيبلغه (مش وقت دوسة زرار Start)
- Push notification (صفارة) لكل الموبايلات لما الراوند يخلص — Cloud Function `dispatchMatchModeAlarms`
- تذكيرات قبل الحدث (24 ساعة / 3 ساعات / ساعة) لكل المسجلين وقت الإرسال — Cloud Function `dispatchEventReminders`
- إشعار للأدمن لما حد يعمل Claim Request (يحاول يربط نفسه بيوزر)

### الصلاحيات (Permissions)
- إصلاح bug: أي عضو (مش بس الأدمن) كان يقدر يعدّل USR الضيوف — بقى للأدمن بس
- Rounds / Breaks / Standings (Closed Individuals) بقوا ظاهرين لكل الأعضاء، مش الأدمن بس — مع حماية كل إجراء إداري (Generate, Regenerate, Swap, تعديل البريك)
- نفس المبدأ اتطبق على Closed Teams (Teams / Breaks / Matches / Standings)

### الواجهة
- أزرار أعضاء الكوميونيتي (Casual/Admin/Remove) بقت خلف قايمة ⋮ مرتبة بدل ما تكون مبعثرة
- الراوندز الخلصانة بقت قابلة للطي (Collapse) تلقائي في CI و CT، لتقصير الشاشة

### Android (Native App)
- تسجيل دخول Google أصلي (Native) بدل الـ WebView popup اللي كان بيفشل
- إشعارات Push أصلية (Native FCM) بدل تقنية المتصفح اللي مش موجودة جوه التطبيق — وبقت تتفعّل تلقائي من غير ما تحتاج تروح Settings
- المشاركة (Share Event) بقت شغالة Native بدل ما تفشل
- دوسة تانية على زرار الباك للخروج فعلياً (مع تحذير بعد أول دوسة)
- الأيقونة، الاسم، ورقم الإصدار مظبوطين على الـ launcher
- صورة البروفايل: fallback تلقائي للحروف الأولية لو الصورة (زي صورة جوجل) فشلت تتحمل
